# Skill: Academic Formatting (/format-latex)

## Purpose
Use this skill whenever generating, formatting, or reviewing academic text, thesis drafts, or LaTeX documents.

## Strict Formatting Rules
1. **Citation Style:** You MUST strictly use `\cite{}` for all citations. Under no circumstances should you use `\parencite{}`. 
2. **Source Selection:** When suggesting, searching for, or integrating academic sources, strongly prioritize recent papers (published within the last 3-5 years) over older, legacy research.
3. **LaTeX Output:** Ensure all generated text is properly escaped and formatted for a standard LaTeX compilation pipeline.

## Execution
When the user types `/format-latex` followed by a prompt, apply all the rules above to the response.
