NBA systems typically rely on structured transaction data, failing to exploit multimodal signals such as text, images, and interaction histories, and lack a unified framework for query-driven retrieval and personalized recommendation. We propose a dual-mode multimodal NBA framework integrating active search and passive personalization in a single architecture. It combines multimodal retrieval with two complementary pipelines: a graph-based component for behavioral patterns and a sequential model for evolving user preferences, jointly modeling semantic relevance, behavioral signals, and temporal dynamics.
On Amazon Books (3.08M items, 100k users), the system achieves HR@10 of 0.7886, outperforming content-based baselines by 81\%, with complementary pipelines recovering each other’s missed predictions.
