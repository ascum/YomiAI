Next Best Action (NBA) systems traditionally rely on structured transaction
data, leaving rich multimedia signals — natural language queries, cover
images, and behavioral traces — underutilized.
We present a dual-mode multimodal NBA recommendation system for large-scale
book discovery that mines text and visual data alongside behavioral
co-occurrence graphs.
The system fuses BGE-M3 text embeddings and CLIP image embeddings via
Reciprocal Rank Fusion for active search, and combines Cleora graph
embeddings with a DIF-SASRec sequential transformer for passive
personalization — all driven by a shared, continuously updated user profile.
Evaluated on a 3.08-million-item Amazon Books catalog with 100k user
histories, DIF-SASRec achieves HR@10 of 0.7749 (+78\% over a content-only
baseline).
The system handles 19 languages via NLLB-200 and serves warm-path queries
at 34\,ms P50.
