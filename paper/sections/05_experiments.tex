\section{Experimental Setup}
\label{sec:exp}

\subsection{Dataset}
We use the Amazon Reviews 2023 Books subset~\cite{hou2024amazon}, filtering
users with fewer than 5 interactions to yield 100k users, 375k graph-covered
items, and 3.08M catalog items (average sequence: 33.5 clicks).
The last item per user is held out; the remainder forms the training set.

\subsection{Evaluation Metrics}
We report HR@10 and NDCG@10.
Following the protocol of SASRec~\cite{kang2018self} and
DIF-SASRec~\cite{xie2022difsr}, each held-out item is ranked against 99 random
negatives (random-chance HR@10 = 0.10).
We also evaluate under a 999-negative protocol (random-chance HR@10 = 0.01)
to assess robustness (Section~\ref{ssec:robustness}).

\subsection{Implementation Details}
All experiments run on a single RTX~5060~Ti (16\,GB).
DIF-SASRec uses AdamW ($\text{lr}=10^{-3}$, wd~$0.01$), linear warmup
(2 epochs), cosine decay to $10^{-5}$ over 30 epochs, batch 2\,048, AMP
float16, 512 sampled softmax negatives, and $\lambda = 0.1$ for the category
loss.
FAISS HNSW uses $M=32$, $\text{ef\_construction}=200$, $\text{ef\_search}=128$.
Cleora uses 3 Markov propagation steps; BGE-M3 and CLIP run in FP16 on GPU;
BM25 on CPU. The AgentPool holds 8 DIF-SASRec instances (148\,MB each,
1.18\,GB total).
